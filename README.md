# AI
AI Related Projects

This project presents a deep learning–based framework for visual weather recognition using outdoor imagery and its integration with smart home automation systems. A convolutional neural network (CNN) is trained on outdoor photographs to classify prevailing weather conditions such as sunrise, rain, shine, and cloudy states. The predicted weather labels are subsequently used as control signals within an Internet of Things (IoT)–enabled smart home environment. Based on the inferred weather conditions, the system autonomously executes predefined actions, including closing windows and disabling automated irrigation systems during rainfall events.
